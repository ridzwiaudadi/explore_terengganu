# Explore Terengganu

## Project Overview

Explore Terengganu is a web-based tourism event portal developed to help tourists and local residents discover and participate in events held throughout Terengganu.

The website allows users to browse event listings, make bookings, manage upcoming reservations, and share reviews based on their experiences. It was designed to provide a centralized platform for tourism event information and simplify the event booking process.

## Technologies Used

* Django 
* HTML
* CSS
* JavaScript

## How to Run the Project

### 1. Clone the repository

```bash
git clone [repository-link]
```

### 2. Navigate to the project folder

```bash
cd explore_terengganu
```

### 3. Install required packages

```bash
pip install django
```

### 4. Run database migrations

```bash
python manage.py migrate
```

### 5. Start the development server

```bash
python manage.py runserver
```

### 6. Open in browser

```
http://127.0.0.1:8000/explore_terengganu/
```
